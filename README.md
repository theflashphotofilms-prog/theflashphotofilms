# Flash Photo Films Website

Welcome to the official website for Flash Photo Films - a professional photography and videography service.

## Project Overview

This is a [Next.js](https://nextjs.org/) project built with the App Router architecture, featuring:

- Modern, responsive design with Tailwind CSS
- Professional portfolio gallery showcasing photography work
- Contact form with API integration
- Full SEO optimization
- Performance-focused image handling
- Mobile-first approach

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Package Manager**: npm

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

Create a `.env.local` file in the root directory and add the following variables:

```bash
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_API_URL=http://localhost:3000/api
NEXT_PUBLIC_ANALYTICS_ID=G-XXXXXXXXXX
CONTACT_EMAIL=info@flashphotofilms.com
```

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop
- **SEO Optimized**: Includes meta tags, structured data, and sitemap support
- **Performance Optimized**: Images are optimized using Next.js Image component
- **Contact Form**: Securely handles form submissions with rate limiting
- **Accessibility**: Built with accessibility best practices in mind

## Deployment

This project is ready to be deployed on Vercel with zero configuration. Simply connect your repository to Vercel and it will automatically deploy on pushes to the main branch.
